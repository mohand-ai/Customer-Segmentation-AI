# نظام تصنيف العملاء الذكي (RFM Analysis & Dashboard) 🚀

## 📝 وصف المشروع
مشروع متكامل يجمع بين تحليل البيانات وتعلم الآلة لتصنيف العملاء.

### 🛠️ التقنيات المستخدمة
* **Python**: (Pandas, Scikit-learn).
* **CustomTkinter**: لتصميم واجهة المستخدم.

## 📊 محتويات المجلد
* `analysis.ipynb`: الكود البرمجي والتحليل الإحصائي.
* `data.csv`: قاعدة البيانات الأساسية.
* `Correct_Customer_Segmentation.xlsx`: التقرير النهائي الملون.
